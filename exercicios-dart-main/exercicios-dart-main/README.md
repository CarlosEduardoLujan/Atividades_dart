A sample command-line application with an entrypoint in `bin/`, library code
in `lib/`, and example unit test in `test/`.
# Exercícios em Dart

Projeto contendo:

- Soma dos números pares
- Fatorial
- Verificação de número primo
- Palíndromo
- Conversão de temperatura
- Cálculo de IMC
- Tabuada

Executar com:
dart run
